# 📱 注册和登录使用指南

## 用户注册步骤

### **第一步：打开小程序**
- 在微信中搜索 "图书馆座位预约系统" 或扫描二维码
- 小程序首次打开时会自动跳转到 **登录页面**

### **第二步：点击去注册**
在登录页面下方，点击 **"还没有账户？去注册"** 链接

![注册入口](./images/register-link.png)

### **第三步：填写注册信息**

需要填写以下信息：

| 字段 | 说明 | 格式要求 |
|------|------|---------|
| 真实姓名 | 你的全名 | 中文或英文，2-50字符 |
| 学号 | 学生ID | 数字和字母组合，建议8-12位 |
| 手机号码 | 联系方式 | 11位数字，以1开头 |
| 设置密码 | 账户密码 | 至少6位，包含字母和数字 |
| 确认密码 | 重新输入密码 | 必须与密码一致 |

**示例：**
```
真实姓名：张三
学号：2022001234
手机号码：13812345678
设置密码：password123
确认密码：password123
```

### **第四步：提交注册**
- 确认所有信息无误后，点击 **"完成注册"** 按钮
- 系统验证信息（通常需要1-2秒）
- 注册成功后，自动跳转到 **座位预约页面**

## 用户登录步骤

### **第一步：打开登录页面**
- 如果已注册，小程序打开时直接进入 **登录页面**
- 如果从其他页面登出，也会自动回到登录页面

### **第二步：输入登录信息**

| 字段 | 说明 |
|------|------|
| 手机号码 | 注册时使用的手机号 |
| 密码 | 注册时设置的密码 |

**示例：**
```
手机号码：13812345678
密码：password123
```

### **第三步：可选 - 记住我**
- 勾选 **"记住我"** 复选框
- 下次登录时会自动填入上次使用的手机号码

### **第四步：点击登录**
- 点击 **"登录"** 按钮
- 系统验证信息（通常需要1-2秒）
- 登录成功后，进入 **座位预约主页**

## 功能详解

### 📝 注册页面功能

| 功能 | 说明 |
|------|------|
| 输入验证 | 实时检查输入格式 |
| 错误提示 | 红色提示框显示错误信息 |
| 成功提示 | 绿色提示框显示成功信息 |
| 去登录 | 如果已有账户，可跳转到登录页 |

#### 常见错误提示

| 错误信息 | 解决方案 |
|---------|--------|
| 请输入手机号码 | 手机号码不能为空 |
| 请输入有效的手机号码 | 手机号码必须是11位，以1开头 |
| 密码长度至少为6位 | 密码需要至少6个字符 |
| 两次输入的密码不一致 | 两次密码输入必须相同 |
| 请输入学号 | 学号不能为空 |
| 既有账户 | 该手机号已注册，请直接登录 |

### 🔐 登录页面功能

| 功能 | 说明 |
|------|------|
| 记住我 | 下次登录时自动填入手机号 |
| 忘记密码 | 联系管理员重置密码 |
| 去注册 | 如果还没有账户，可跳转到注册页 |

#### 常见错误提示

| 错误信息 | 解决方案 |
|---------|--------|
| 请输入手机号码 | 手机号码不能为空 |
| 请输入有效的手机号码 | 手机号码格式错误 |
| 请输入密码 | 密码不能为空 |
| 登录失败，请重试 | 手机号或密码错误 |
| 网络连接失败 | 检查网络连接 |

## 安全建议

### ✅ 建议做法

- ✓ 使用强密码（混合大小写字母、数字和符号）
- ✓ 在私人设备上使用"记住我"功能
- ✓ 定期修改密码
- ✓ 尽量不在公共设备上登录

### ❌ 不建议做法

- ✗ 使用简单密码（如12345678、000000等）
- ✗ 在公共设备上勾选"记住我"
- ✗ 将账户信息分享给他人
- ✗ 在不安全的网络上登录

## 问题排查

### Q1: 注册后无法登录？
**A:** 
1. 确认手机号和密码输入正确
2. 检查网络连接是否正常
3. 尝试重启小程序后再登录
4. 如仍无法登录，联系管理员

### Q2: 密码忘记了怎么办？
**A:**
1. 在登录页点击 **"忘记密码？"**
2. 按照指示操作（通常需要验证身份）
3. 或直接联系图书馆管理员

### Q3: 手机号已注册，但无法登录？
**A:**
1. 确认密码是否正确
2. 检查是否开启了VPN或代理
3. 尝试在其他WiFi网络下登录
4. 联系管理员进行账户检查

### Q4: 注册失败提示"既有账户"？
**A:**
该手机号已被注册，直接使用该手机号登录即可

### Q5: 页面加载很慢？
**A:**
1. 检查网络连接质量
2. 关闭其他占用网络的应用
3. 尝试切换到其他WiFi或移动数据
4. 重启小程序
