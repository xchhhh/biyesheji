# 🎯 用户注册和登录快速指南

## 📱 用户在哪里注册？

### 方式一：通过小程序首页
1. **打开小程序** → 看到登录页面
2. **点击"还没有账户？去注册"** → 进入注册页面
3. **填写注册信息**：
   - 真实姓名（必填）
   - 学号（必填）
   - 手机号码（必填，11位）
   - 设置密码（必填，至少6位）
   - 确认密码（必填，需与密码一致）
4. **点击"完成注册"** → 自动登录并进入座位预约页面

### 方式二：通过Web网页
- 访问注册页面 URL（如果有配置）
- 填写相同的信息后注册

| 字段 | 说明 | 示例 | 规则 |

## 🔐 登录流程

### 首次登录（注册后自动）
- 注册完成后自动登录（不需要再输入密码）

### 后续登录
1. **打开小程序** → [登录页面]
2. **输入信息**：
   - 手机号码：13812345678
   - 密码：password123
3. **点击"登录"** → 进入主页

### 登出后重新登录
- 在个人中心/设置中 → 点击"退出登录"
- 然后按照"后续登录"步骤重新登录

## 🛡️ 注册时常见错误

| 错误信息 | 原因 | 解决办法 |
|---------|------|--------|
| 请输入手机号码 | 手机号为空 | 填入有效的11位手机号 |
| 请输入有效的手机号码 | 手机号格式不对 | 确保是11位，以1开头 |
| 密码长度至少为6位 | 密码太短 | 至少输入6个字符 |
| 两次输入的密码不一致 | 两次密码不同 | 确保两次密码完全相同 |
| 请输入学号 | 学号为空 | 输入你的学号 |
| 既有账户 | 该手机号已注册 | 用此手机号直接登录 |
| 既有账户 | 学号已被注册 | 联系管理员 |

---

| 错误信息 | 原因 | 解决办法 |
|---------|------|--------|

## 📞 需要帮助？

### 常见问题

**Q: 忘记了我的学号怎么办？**  
A: 登录小程序后在个人信息中查看，或直接联系学校教务处

**Q: 可以用其他手机号创建第二个账户吗？**  
A: 可以，每个手机号对应一个账户，但学号必须唯一

**Q: 密码可以更改吗？**  
A: 可以，在个人中心/账户设置中修改密码

**Q: 账户信息可以修改吗？**  
A: 除学号外其他信息可修改，学号无法修改（可联系管理员）

## 🔄 Token 管理

### 获取 Token
- 注册或登录时自动获得
- Token 保存在小程序本地存储中
- Token 用于后续 API 调用的身份验证

### Token 使用
```javascript
// 获取 token
const token = wx.getStorageSync('token')

// 在请求头中使用
{
  'Authorization': `Bearer ${token}`,
  'Content-Type': 'application/json'
}
```

### Token 过期
- Token 有有效期（默认24小时）
- 过期需重新登录
- 系统会自动提示重新登录
