## 🐛 如果遇到问题

### 问题 1：网络错误 / Cannot connect

**解决：**
1. 检查后端是否还在运行（PowerShell 窗口中是否有 Flask 输出）
2. 在另一个 PowerShell 窗口运行：
   ```powershell
   $body = '{"phone_number":"13800000001","password":"test123"}'
   Invoke-WebRequest -Uri "http://127.0.0.1:5000/api/auth/register" -Method POST -Headers @{"Content-Type"="application/json"} -Body $body -UseBasicParsing
   ```
3. 如果出现状态码 201 或 400，说明后端正常，问题在于小程序配置

### 问题 2：显示白屏或错误

**解决：**
1. 按 F12 打开开发者工具
2. 查看 Console 标签页是否有红色错误
3. 检查 Network 标签页看是否有请求失败

### 问题 3：跨域错误 (CORS)

**解决：**
- 确保在微信开发者工具的 **详情** 中勾选了 **"不校验合法域名"**
- 这个选项很重要！
