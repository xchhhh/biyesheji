## 💡 常见操作指南

### 1. 禁用违规用户

1. 登录管理后台
2. 点击侧边栏 → 用户管理
3. 找到目标用户
4. 点击【禁用】按钮
5. 输入禁用原因
6. 确认

**效果**:
- 用户账户被禁用
- 用户的所有待签到预约被取消
- 操作被记录到审计日志

### 2. 报告座位损坏

1. 点击侧边栏 → 座位维护
2. 点击【报告问题】按钮
3. 填写表单:
   - 座位ID
   - 问题类型（破损/脏污/家具/电气/其他）
   - 严重程度（低/中/高/严重）
   - 问题描述
4. 点击【提交】

**效果**:
- 座位状态被标记为"维修"
- 生成维护记录
- 用户无法预约该座位

### 3. 发布系统公告

1. 点击侧边栏 → 公告管理
2. 点击【发布新公告】按钮
3. 填写表单:
   - 标题: "系统维护通知"
   - 内容: 详细说明
   - 类型: 维护/紧急/一般
   - 优先级: 低/中/高
   - 勾选"置顶"使公告显示在最前面
4. 点击【发布】

**效果**:
- 公告立即发布
- 用户端会显示该公告
- 操作被记录

### 4. 查看座位实时占用

1. 点击侧边栏 → 值班快查
2. 查看所有房间的占用情况:
   - 总座位数
   - 占用座位数
   - 空闲座位数
   - 维修座位数
   - 占用率百分比
3. 点击【详细信息】查看单个房间的座位分布

**显示信息**:
- 实时更新的座位占用状态
- 每个座位的使用者
- 颜色编码（绿=空闲, 黄=占用, 红=维修）

### 5. 查看审计日志

1. 点击侧边栏 → 审计日志
2. 选择过滤条件:
   - 模块: 用户/维护/公告/预约
   - 状态: 成功/失败
3. 查看日志列表，包括:
   - 操作时间
   - 操作人
   - 操作类型
   - 操作描述
   - 资源类型和ID

**用途**:
- 追溯管理操作
- 合规性检查
- 问题排查

## 📈 数据看板指标解释

### 系统用户卡片
- **总用户**: 系统中的所有用户数量
- **活跃**: 状态为正常的用户数
- **禁用**: 被禁用的用户数

### 今日预约卡片
- **总预约**: 今天的所有预约数
- **签到**: 今天已签到的预约
- **完成**: 今天已完成的预约

### 座位状态卡片
- **总座位**: 系统中的座位总数
- **占用**: 当前被使用的座位数
- **维修**: 当前处于维修状态的座位数

### 待处理维护卡片
- 需要维修人员处理的座位问题数量
- 实时更新

## 🐛 故障排除

### 问题：无法登录
**解决方案**:
1. 确保服务器正在运行
2. 检查用户名是否为 "root"
3. 检查密码是否为 "123456"
4. 清除浏览器缓存重试

### 问题：页面加载缓慢
**解决方案**:
1. 检查网络连接
2. 刷新页面
3. 关闭浏览器其他标签页
4. 检查服务器日志

### 问题：数据不更新
**解决方案**:
1. 按F5刷新页面
2. 检查浏览器控制台是否有错误
3. 检查API是否正常响应
4. 重启服务器

### 常见问题
Q: 如何重置管理员密码?
A: 当前为测试环境，固定为 root / 123456。生产环境需要实现密码重置功能。

Q: 可以添加多个管理员吗?

Q: 日志保留多久?
A: 默认查询显示最近7天，可根据需要调整。
