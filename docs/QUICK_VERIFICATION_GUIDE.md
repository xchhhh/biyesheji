## 🐛 常见问题排查

### 问题: 预约后座位颜色没有改变

```
检查项:
□ 代码是否已保存 (Ctrl+S)
□ 小程序是否已重新加载 (Ctrl+R)
□ 浏览器 Console 是否有红色错误
□ updateLocalSeatStatus() 方法是否存在
□ onConfirmReservation() 中是否调用了 updateLocalSeatStatus()

如果都检查了仍有问题:
  1. 查看 Console 找出具体错误
  2. 检查 seats.js 中的 updateLocalSeatStatus 方法
  3. 确认 seatId 匹配正确
```

### 问题: 热力表没有显示

```
检查项:
□ 代码是否已保存 (Ctrl+S)
□ 小程序是否已重新加载 (Ctrl+R)
□ 是否向下滚动到了座位网格下方
□ data 中的 showHeatmap 是否为 true
□ WXML 中是否包含 heatmap-section
□ WXSS 中是否包含热力表样式

如果都检查了仍有问题:
  1. 在 Browser DevTools 中检查 DOM 结构
  2. 搜索是否存在 class="heatmap-section"
  3. 查看是否被 CSS 隐藏或遮挡
```

### 问题: 热力表显示但样式不对

```
检查项:
□ 颜色是否显示 (应该是渐变色)
□ Emoji 图标是否显示 (🔥 ⭐ ❄️)
□ 文字是否对齐
□ 是否超出屏幕边界

解决方案:
  1. 检查 seats.wxss 中热力表样式是否完整
  2. 清除浏览器缓存 (Ctrl+Shift+Delete)
  3. 重新加载小程序 (Ctrl+R)
  4. 在开发者工具中检查 DOM 的 Style
```
